# QR Scan Tracker

Dynamic QR code tracker with real-time scan analytics built on Cloudflare Workers.

## Features
- Create and manage dynamic QR codes per event
- Real-time scan tracking via Cloudflare Workers KV
- Event management with multiple QR codes per event
- Used for concert flyers and non-profit event marketing

## Stack
HTML, JavaScript, Cloudflare Workers, Supabase

## Status
Live but Local — not publicly deployed
